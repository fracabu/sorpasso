<a name="english"></a>
## :gb: English

### Overview

Landing page for "Il Sorpasso" car rental service. One-page responsive design with contact form that sends requests via Firebase Functions.

### Features

- **Responsive Landing Page** - Hero, Services, Gallery, Contacts sections
- **Contact Form** - Writes to Firestore, sends email via Cloud Function
- **Floating WhatsApp** - Quick chat access
- **Multilingual** - Italian/English support with vue-i18n

### Sections

- Hero with call-to-action
- Services showcase
- Photo gallery
- Contact form with validation
- WhatsApp floating button

### Quick Start

```bash
git clone https://github.com/fracabu/sorpasso.git
cd sorpasso
npm install
npm run dev
```

## :it: Italiano

### Panoramica

Landing page per il servizio di noleggio auto "Il Sorpasso". Design responsive one-page con form di contatto che invia richieste via Firebase Functions.

### Funzionalita

- **Landing Page Responsive** - Sezioni Hero, Servizi, Gallery, Contatti
- **Form di Contatto** - Scrive su Firestore, invia email via Cloud Function
- **WhatsApp Floating** - Accesso chat rapido
- **Multilingua** - Supporto italiano/inglese con vue-i18n

### Sezioni

- Hero con call-to-action
- Vetrina servizi
- Galleria foto
- Form contatto con validazione
- Bottone WhatsApp floating

### Avvio Rapido

```bash
git clone https://github.com/fracabu/sorpasso.git
cd sorpasso
npm install
npm run dev
```
